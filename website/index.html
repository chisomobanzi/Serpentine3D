Precision 3D for the things you build.

Serpentine3D is a free modelling and drafting tool with the exactness of Rhino — true curves, real dimensions, and drawing sheets a workshop can build from. Made by one person who needs it, not a software company.

DETAIL A — GEOMETRY

Curves that measure true

Surfaces here are real mathematics, not triangles — the same NURBS geometry Rhino uses. A 300 mm radius is a 300 mm radius at any zoom, on any drawing. It opens and saves Rhino .3dm files (curves exact, layers preserved), and exports STEP, DXF, glTF and USD for the rest of your pipeline.

DETAIL B — DRAFTING

From model to drawing sheet

Paper-space sheets from A4 to A0 with live detail views at real scales — 1:25, 1:50 — plus hidden-line technical views, dimensions, leaders, hatching, scale bars, title blocks and revision tables. Print straight to PDF and hand it to the workshop.

DETAIL C — HANDS

Your Rhino muscle memory works

A command line that speaks Rhino — move, loft, fillet, offset — with the same aliases, object snaps and a gumball for direct manipulation. If your hands know Rhino, they already know this. If they don't, there's a menu for everything.

DETAIL D — ASSISTANT

Model with an AI that can see the screen

Serpentine speaks MCP, so an assistant like Claude can look at your viewport and run any tool you can — draw a curve, loft a surface, run a boolean, measure a span, export a STEP. Describe the change and watch it happen, or hand off the repetitive parts and keep designing. No other CAD tool is built to be operated this way.
